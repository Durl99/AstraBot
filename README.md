## 🚀 ¿Qué es AstraBot?

AstraBot no busca verse como un bot genérico.

Su objetivo es mezclar:

- 🛡️ **control real para grupos**
- 💰 **sistemas que mantengan actividad**
- 💞 **interacción social y divertida**
- 🌌 **una personalidad propia y reconocible**

La idea es que el grupo no solo tenga utilidades, sino también una **experiencia con carácter**.

---

## 🌠 Características principales

### 🛡️ Moderación astral
- Anti-link
- Anti-flood
- Warnings
- Kick / Ban / Unban
- Promote / Demote
- Abrir / cerrar grupo
- Ghost tag, hidden tag y herramientas para admins

### 💰 Economía galáctica
- Coins
- Banco

### 💞 Interacción social

### 🪐 Media & utilidades

## 📦 Instalación local

### 1. Clona el repositorio

```bash
git clone https://github.com/TUUSUARIO/AstraBot.git
cd AstraBot
```

### 2. Instala dependencias

```bash
npm install
```

### 3. Configura tu entorno

Crea un archivo `.env`:

```env
BOT_NAME=AstraBot
PREFIX=.
OWNER_NUMBER=506XXXXXXXX
OWNER_LID=TU_LID_SI_LA_USAS
```

### 4. Inicia el bot

```bash
npm start
```

---

## 🔐 Seguridad

**No subas nunca** estos archivos al repositorio:

```txt
.env
src/sessions
node_modules
```

Usa un `.gitignore` correcto para proteger:
- variables sensibles
- sesiones de WhatsApp
- dependencias innecesarias
