## ✨ ¿Qué es AstraBot?

**AstraBot** no busca ser un bot genérico.  
Está diseñado para ofrecer una experiencia más viva dentro de grupos de WhatsApp, mezclando:

- 🛡️ herramientas de moderación
- 🌠 automatización útil
- 💰 economía e inventario
- 🎮 diversión e interacción social
- 🚀 una personalidad astral, espacial y dominante

Su objetivo es que el grupo no solo tenga utilidades, sino también una **identidad más fuerte y entretenida**.

## 🧭 Filosofía del bot

AstraBot está hecho para que cada comando:
- se sienta con **presencia**
- tenga una **estética espacial**
- y haga que el grupo viva una experiencia más única

No es solo “responder comandos”.  
La idea es que el bot tenga **carácter**.
